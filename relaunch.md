 Patterns from the Best

 ### 1. Linear — The Gold Standard

 - Dark theme, product UI dominates the hero
 - Bento grid feature tiles with real screenshots in each
 - Headline under 8 words: "The product development system for teams and agents"
 - Zero abstract illustrations — just the product

 ### 2. Cursor — Social Proof Mastery

 - Hero shows actual IDE interface with agent interactions
 - Quotes from Jensen Huang, Andrej Karpathy, Patrick Collison
 - Interactive demos embedded directly in the page
 - "Cursor is your coding agent" — crystal clear positioning

 ### 3. Vercel — Metrics + Trust

 - Customer logos paired with specific metrics: "build times went from 7m to 40s"
 - Animated gradients, code snippets side-by-side with visuals
 - Clean card-based feature sections

 ### 4. Cognition — Editorial Minimalism

 - Numbered sections (01, 02, 03...)
 - Ultra-minimal nav (5 links)
 - Customer logo wall
 - Bold, confident typography

 ### 5. Replit — Interactive Hero

 - "What will you build?" with prompt input in hero
 - Strong testimonials with photos + titles
 - Feature cards with clear icons + descriptions
 - Pricing table for conversion
