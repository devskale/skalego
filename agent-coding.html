Agentic Coding.

Neben Kundenprojekten entsteht hier Open-Source-Arbeit für AI-Coding-Agenten. Im Mittelpunkt steht skale-skills — ein Paket mit Skills, Extensions und Prompts — plus eine wachsende Sammlung an Best-Practice-Guides für Agenten-Entwicklung.

devskale/skale-skills

Ein pi-Paket mit Skills, Extensions und Prompts für AI-Coding-Agenten. Funktioniert mit pi, Claude Code, Codex, OpenCode und anderen. Skills decken Web-Suche, Web-Extraktion, YouTube, Headless-Browser-Automatisierung, TODO-Tracking und mehr ab — plus Werkzeuge zum Erstellen eigener Skills und AGENTS.md-Dateien.

★ Open Source · pi / Claude Code / Codex · Skills · Extensions · Prompts
install
# Alles installieren (pi)
pi install git:github.com/devskale/skale-skills

# Oder erst ausprobieren, ohne zu installieren
pi -e git:github.com/devskale/skale-skills
